MyBot - A Simple Chatbot Interface

Welcome to MyBot! This is a simple chatbot interface created using HTML and CSS. You can type your messages in the input field below and click the "Send" button to interact with the bot.

The bot was created on Botpress. The bot will answer questions using a database that holds data from a business's sales transactions.

This bot can only answer questions about Top 10 customers by total orders placed.

Click on chat icon in the bottom right corner to start chatting with MyBot!
